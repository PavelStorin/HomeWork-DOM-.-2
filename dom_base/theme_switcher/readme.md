# DOM Practice — Theme Switcher

Небольшая задача для практики **работы с классами элементов и переключения состояний в DOM**.

Цель — реализовать переключение между **светлой и тёмной темой страницы**.

---

# 📋 Задача

На странице есть кнопка **Toggle theme**.

Необходимо реализовать JavaScript-логику, которая позволит:

- переключать тему страницы
- менять стили с помощью CSS-переменных

---

# 🧱 Структура страницы

Главные элементы:

| Элемент      | Назначение               |
| ------------ | ------------------------ |
| `.container` | контейнер контента       |
| `#themeBtn`  | кнопка переключения темы |

HTML страницы:

```html
<div class="container">
  <h1>Theme switcher</h1>

  <button id="themeBtn">Toggle theme</button>
</div>
```

---

# ✅ Требования

При клике на кнопку:

1. должен добавляться или удаляться класс `dark`
2. стили страницы должны изменяться

---

# 🎨 Как работают стили

В CSS используются **переменные (`CSS variables`)**.

По умолчанию используются светлые цвета:

```css
:root {
  --bg-color: #ffffff;
  --text-color: #222;
}
```

Страница использует эти переменные:

```css
body {
  background: var(--bg-color);
  color: var(--text-color);
}
```

Когда добавляется класс `dark`, значения переменных меняются:

```css
.dark {
  --bg-color: #1e1e1e;
  --text-color: #ffffff;
}
```

Из-за этого вся страница автоматически меняет тему.
